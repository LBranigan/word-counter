  • Google OAuth 2.0
  • Session management
  • User profile display
  • Sign-out handling

🎤 Audio Recording

  • MediaRecorder API
  • 30s / 1min / 2min options
  • WebM/Opus format
  • Mobile codec fallback

📷 Image Capture

  • getUserMedia API
  • HD resolution (1920x1080)
  • File upload alternative
  • Canvas rendering

  • Touch/mouse selection

Project File Structure
word-analyzer/ ├── index.html ← Main HTML (~400 lines) ├── styles.css ← Responsive CSS (~2000 lines) ├── app.js ← Core logic (~5000 lines) ├── firebase-auth.js ← Authentication (~150 lines) ├── firebase-db.js ← Database ops (~350 lines) ├── firebase-config.js ← Firebase keys (not in repo) └── manifest.md ← Documentation (~800 lines)
Assessment Output Metrics

📈 Accuracy

  • Correct words / Total words
  • Percentage score
  • Color-coded (green/yellow/red)

⚡ WPM

  • Words per minute
  • Based on audio duration
  • Grade-level benchmarks

🎵 Prosody

  • 1.0 - 4.0 scale
  • Expression & rhythm
  • Based on pauses & pacing

📋 Error Analysis

  • Phonics patterns
  • Reading strategies
  • Speech patterns
  • Recommendations
