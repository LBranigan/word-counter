Design Notes:

  • Extremely subtle: Pills appear as lightweight metadata tags at the bottom
  • Minimal visual weight: Soft gray colors (#f1f3f5 background, #6c757d text)
  • Hover interaction: Each pill shows a tooltip with full details on hover
  • Short labels: Each pill has 2-3 words max for clean appearance
  • Icon-enhanced: Small emoji icons provide visual context without adding weight
  • Non-intrusive: Separated by a subtle border, feels like "footnotes"
  • Responsive tooltips: Dark tooltips with smooth animation for readability
  • Professional polish: Rounded corners, subtle hover states, clean typography
