Setup Required

This app uses Google Cloud Vision API for highly accurate word detection (99%+ accuracy) and Speech-to-Text API for audio analysis.

Quick Setup (2 minutes):

  1. Go to Google Cloud Console 🔗
  2. Create a new project (or select an existing one)
  3. Create a new API key under "Credentials"
  4. Enable "Cloud Vision API" (for text recognition)
  5. Enable "Cloud Speech-to-Text API" (for audio analysis)
  6. Copy your API key and paste it below

✅ Free tier: 1,000 Vision requests + 60 minutes Speech/month

🔒 Your API key will be validated and securely synced to your Google account

📖 Need detailed help? See the Setup Guide

🔐 Your API key is encrypted and synced to your Firebase account. You only need to set this up once.
